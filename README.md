# odin-recipes
Project Recipes
This is my first structured HTML project, built as part of The Odin Project's Front-End Developer course. It's a simple recipe website made up of multiple linked HTML pages with each containing a dish image, description, ingredients, and preparation steps.
Through this project, I have demonstrated my understanding of core HTML concepts including page structure, headings, paragraphs, lists, images, and anchor links. It marks an important step in my journey from teaching HTML and CSS to actually building things with them myself.
